# Icon

Renders an icon from a collection.

## Usage

:::code-group
```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
// ---cut---
import { createSystem } from 'frog/ui'

const { Icon } = createSystem()

function Example() {
  return <Icon name="zap" />
}
```

## `Icon` Properties

### `color`

Sets the color of the icon. Defaults to `'currentColor'` or the color of the icon (when [`mode`](#mode) is set to `'bg'`).

:::code-group
```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
import { createSystem } from 'frog/ui'

const { Icon } = createSystem()

function Example() {
  return (
// ---cut---
<Icon
  color="green800" // [!code focus]
//       ^|
  name="zap"
/>
// ---cut-after---
  )
}
```

This prop is only supported when [`mode`](#mode) is `'mask'` or `'auto'` (and the icon with [`name`](#name) is inferred as `'mask'`).
:::

function Example() {
  return (
// ---cut---
<Icon
  name="zap"
  mode="auto" // [!code focus]
// ^?
/>

  )
}

### `collection`

Icon collection to use for resolving icons. Defaults to `'lucide'`.

The following collections are available:
- [Heroicons](https://heroicons.com)
- [Lucide](https://lucide.dev)
- [Radix Icons](https://www.radix-ui.com/icons)

Collection is mapped to the [`icons` property](/ui/ui-system#icons) on the UI System Variables.

:::code-group
```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
import { createSystem } from 'frog/ui'

const { Icon } = createSystem()

function Example() {
  return (
// ---cut---
<Icon
  name="bolt"
  collection="heroicons" // [!code focus]
// ^?
/>


// ---cut-after---
  )
}
```
