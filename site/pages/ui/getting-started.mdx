# Getting Started

On this page, we will guide you through the process of setting up a UI System for your frames in Frog.

## Steps

::::steps

### Create your UI System

Firstly, we will need to create our [UI System](#TODO) with `createSystem`. 
The `createSystem` function takes in a tokens object, and returns a set of primitive components to build our frame UI.

By default, if no tokens are passed to `createSystem`, the system will use the default tokens.

```tsx twoslash [src/ui.ts]
import { createSystem } from 'frog/ui'

export const { Box, Heading, Text, VStack, tokens } = createSystem()
```

[Read more on Tokens](#TODO) to learn how to customize the system.

:::tip[Tip]
You can also control the color scheme of the system by passing in a `colors` property to `createSystem`. Frog supports a `light` and `dark` color scheme.

```tsx twoslash [src/ui.ts]
import { colors, createSystem } from 'frog/ui'

export const { Box, Heading, Text, VStack, tokens } = createSystem({
  colors: colors.light, // [!code focus]
})
```
:::

### Consume the System

Once we have our system set up, we will need to inject the `tokens` into our `Frog` instance.

After that, we can then consume our components to build our frame UI.

:::code-group

```tsx twoslash [src/index.ts]
import { Frog } from 'frog'
import { Box, Heading, Text, VStack, tokens } from './ui.js'

export const app = new Frog({
  tokens, // [!code focus]
}).frame('/', (c) => {
  return c.res({
    image: (
      <Box grow alignHorizontal="center" alignVertical="center"> // [!code focus]
        <VStack gap="4"> // [!code focus]
          <Heading>FrogUI üê∏</Heading> // [!code focus]
          <Text color="text200" size="20"> // [!code focus]
            Build consistent frame experiences // [!code focus]
          </Text>  // [!code focus]
        </VStack> // [!code focus]
      </Box> // [!code focus]
    )
  })
})
```

```tsx twoslash [src/ui.ts] filename="./ui.js"
import { createSystem } from 'frog/ui'

export const { Box, Heading, Text, VStack, tokens } = createSystem()
```
