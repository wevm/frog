# VStack

Renders children vertically, with consistent spacing (if specified)
between them.

## Usage

:::code-group

```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
// ---cut---
import { createSystem } from 'frog/ui'

const { Box, VStack } = createSystem()

function Example() {
  return (
    <VStack gap="8" grow> // [!code focus]
      <Box backgroundColor="red" height="100%" /> // [!code focus]
      <Box backgroundColor="red" height="100%" /> // [!code focus]
      <Box backgroundColor="red" height="100%" /> // [!code focus]
    </VStack> // [!code focus]
  )
}
```

<div data-title="Preview">
  <div style={{ display: 'flex', flexDirection: 'column', gap: 8, height: '100px' }}>
    <div style={{ backgroundColor: 'red', flex: 1, width: '100%' }} />
    <div style={{ backgroundColor: 'red', flex: 1, width: '100%' }} />
    <div style={{ backgroundColor: 'red', flex: 1, width: '100%' }} />
  </div>
</div>

:::

## `VStack` Properties

### `alignHorizontal`

Horizontally aligns the contents.

:::code-group

```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
import { createSystem } from 'frog/ui'

const { VStack } = createSystem()

function Example() {
  return (
// ---cut---
<VStack 
  alignHorizontal="center" // [!code focus]
//                 ^| 
  grow
>
// ---cut-after---
    </VStack> 
  )
}
```

<div data-title="Preview">
  <div style={{ alignItems: 'center', display: 'flex', flexDirection: 'column', gap: 8, height: '100px' }}>
    <div style={{ backgroundColor: 'red', flex: '1', width: '30px' }} />
    <div style={{ backgroundColor: 'red', flex: '1', width: '30px' }} />
    <div style={{ backgroundColor: 'red', flex: '1', width: '30px' }} />
  </div>
</div>

:::
