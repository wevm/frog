# FrogUI

**FrogUI** is an extension of the Frog Framework that provides layout & styling primitives to help you build extensible and consistent dynamic [Frame Image](/concepts/images-intents#image) interfaces.

Create your [UI System](#TODO) with optional theming tokens, and build your frames with [Primitive Components](#TODO).

:::code-group

```tsx [index.ts]
import { Frog } from 'frog'
import { Box, Heading, Text, VStack, tokens } from './ui.js'

export const app = new Frog({
  tokens,
}).frame('/', (c) => {
  return c.res({
    image: (
      <Box grow alignHorizontal="center" alignVertical="center">
        <VStack gap="4">
          <Heading>FrogUI ğŸ¸</Heading>
          <Text color="text200" size="20">
            Build consistent frame experiences
          </Text>
        </VStack>
      </Box>
    )
  })
})
```
