# FrogUI

**FrogUI** is an extension of the Frog Framework that provides type-safe layout & styling primitives to help you build extensible and consistent dynamic [Frame Image](/concepts/images-intents#image) interfaces.

It is a **systematic & type-safe alternative** to styling components using the Tailwind (`tw`) or `style` property of [Satori](https://github.com/vercel/satori).

Create your [UI System](/ui/ui-system) with optional [variables](/ui/ui-system#ui-variables), and build your frames with [Primitive Components](/ui/primitive-components).
