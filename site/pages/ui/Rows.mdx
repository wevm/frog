# Rows

Renders children horizontally, with consistent spacing (if specified)
between them. Rows can also have a specific `height` value.

## Usage

:::code-group

```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
// ---cut---
import { createSystem } from 'frog/ui'

const { Rows, Row } = createSystem()

function Example() {
  return (
    <Rows gap="8" grow> // [!code focus]
      <Row backgroundColor="red" height="1/4" /> // [!code focus]
      <Row backgroundColor="red" height="2/4" /> // [!code focus]
//                                        ^|
      <Row backgroundColor="red" height="1/4" /> // [!code focus]
    </Rows> // [!code focus]
  )
}
```

<div data-title="Preview">
  <div style={{ display: 'flex', flexDirection: 'column', gap: 8, height: '200px' }}>
    <div style={{ backgroundColor: 'red', flex: 1 }} />
    <div style={{ backgroundColor: 'red', flex: 2 }} />
    <div style={{ backgroundColor: 'red', flex: 1 }} />
  </div>
</div>

:::

## `Rows` Properties

### `alignHorizontal`

Horizontally aligns the contents.

:::code-group

```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
import { createSystem } from 'frog/ui'

const { Rows, Row } = createSystem()

function Example() {
  return (
// ---cut---
<Rows 
  alignHorizontal="center" // [!code focus]
//                 ^| 
  grow
>
// ---cut-after---
      <Row backgroundColor="red" width="100%" height="1/4" /> // [!code focus]
      <Row backgroundColor="red" width="100%" height="2/4" /> // [!code focus]
      <Row backgroundColor="red" width="100%" height="1/4" /> // [!code focus]
    </Rows> // [!code focus]
  )
}
```

<div data-title="Preview">
  <div style={{ justifyContent: 'center', display: 'flex', flexDirection: 'column', gap: 8, height: '200px' }}>
    <div style={{ backgroundColor: 'red', flex: '0.25', height: '100%' }} />
    <div style={{ backgroundColor: 'red', flex: '0.25', height: '100%' }} />
  </div>
</div>

:::

## `Row` Properties

### `height`

Sets the height span of the Row (in fractions).

:::code-group

```tsx twoslash [Code]
/** @jsxImportSource frog/jsx */
// ---cut---
import { createSystem } from 'frog/ui'

const { Rows, Row } = createSystem()

function Example() {
  return (
    <Rows gap="8" grow> 
      <Row backgroundColor="red" width="100%" height="1/4" /> 
      <Row backgroundColor="red" width="100%" height="2/4" /> // [!code focus]
//                                                     ^|
      <Row backgroundColor="red" width="100%" height="1/4" /> 
    </Rows> 
  )
}
```

<div data-title="Preview">
  <div style={{ display: 'flex', flexDirection: 'column', gap: 8, height: '200px' }}>
    <div style={{ backgroundColor: 'gray', flex: 1, height: '100%' }} />
    <div style={{ backgroundColor: 'red', flex: 2, height: '100%' }} />
    <div style={{ backgroundColor: 'gray', flex: 1, height: '100%' }} />
  </div>
</div>

:::

### `Box` Properties

`Rows` inherits the [properties of `Box`](/ui/box).
