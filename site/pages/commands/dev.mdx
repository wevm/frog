# dev

Start development server at entrypoint.

## Usage

```bash
frog dev [entrypoint]
```

## Args

### path

`string`

Path to Frog entrypoint. Defaults to `(src|api)/index.{js,jsx,ts,tsx}`.

```bash
frog dev src/frog.tsx
```

### options

#### -h, --host

`boolean`

Expose host URL.

```bash
frog dev --host
```

#### -p, --port

`number`

Port used by the server. Defaults to `5173`.

```bash
frog dev --port 3001
```
